# **Realtime Chat App**

A **real-time chat application** built using **Node.js**, **Express**, and **Socket.IO**. Users can join with a username, send messages instantly, and see emojis as avatars. The interface uses **pastel colors** for a clean, modern look.
---
## **Tech Stack**

* **Backend:** Node.js, Express, Socket.IO
* **Frontend:** HTML, CSS, JavaScript
* **Real-Time Communication:** WebSockets via Socket.IO
---
## **Features**

* Enter a username to join the chat
* Real-time messaging across multiple users
* Unique emoji avatar per user
* Pastel color UI for messages
* Simple and responsive design
