# **Realtime Chat App**

A **real-time chat application** built using **Node.js**, **Express**, and **Socket.IO**. Users can join the chat with a username, send messages instantly, and interact in real time. The application features **emoji avatars** and a **pastel-colored UI** for a clean and modern user experience.

---

## **Tech Stack**

* **Backend:** Node.js, Express, Socket.IO
* **Frontend:** HTML, CSS, JavaScript
* **Real-Time Communication:** WebSockets (Socket.IO)

---

## **Features**

* Join the chat using a username
* Instant real-time messaging between multiple users
* Automatic emoji avatar assignment
* Clean pastel-themed user interface
* Simple and responsive design
